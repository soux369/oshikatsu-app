プライバシーポリシー

最終更新日: 2026年2月4日

1. はじめに

「あおぎり通知（非公式）」（以下、「本アプリ」）は、VTuberグループ「あおぎり高校」の配信情報を通知するアプリケーションです。本ポリシーは、本アプリにおける利用者情報の取り扱いについて説明するものです。

2. 収集する情報とその利用目的

本アプリは、以下の情報を収集・利用する場合があります。

  • プッシュ通知用トークン (Expo Push Token):
    利用者の端末に配信開始通知などを送信するために使用します。このトークンは特定の個人を識別するものではなく、通知送信以外の目的では使用されません。

3. 外部サービスの利用

本アプリは、情報の取得や機能提供のために以下の第三者サービスを利用しています。

YouTube Data API

配信情報の取得に Google社の YouTube Data API を使用しています。
YouTubeのコンテンツを表示することで、利用者は YouTube 利用規約 (https://www.youtube.com/t/terms) に同意したものとみなされます。
また、Google プライバシーポリシー (http://www.google.com/policies/privacy) に基づいて情報が取り扱われます。

Expo (EAS)

プッシュ通知の配信基盤として Expo のサービスを利用しています。

4. 第三者への提供

法令に基づく場合を除き、取得した個人情報を利用者の同意なく第三者に提供することはありません。

5. 免責事項

本アプリはファンが作成した非公式アプリであり、あおぎり高校公式および所属事務所とは一切関係ありません。本アプリの利用により生じた損害について、開発者は一切の責任を負いません。
